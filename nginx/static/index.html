Python Celery DEMO

You can reach the following services here:

  • Celery Flower (montiroing tool for celery)
  • RabbitMQ management interface (username: guest, password: guest)
  • MongoDB management interface (username: guest, password: guest)

Creating a new task

You can start a new task by posting to the http://localhost:8080/api/tasks/>task name< The following tasks available

Long add

Itt adds two numbers and waits a minute. 

    POST http://localhost:8080/api/tasks/long-add
    Content-Type: application/json

    { "a": 4, "b": 5 }

Fibonacci

It calcualtes the nth fibonacci number. 

    POST http://localhost:8080/api/tasks/fibonacci
    Content-Type: application/json

    { "n": 10 }

Getting info from the task

You can get info of the not finished tasks by sending GET request to http://localhost:8080/api/tasks/>task_id< URI. The task ID is returned as a response of the POST requests or it can be read in the Celery Flower UI. 

    GET http://localhost:8080/api/tasks/cd2abce4-06a1-460a-a042-570fa0bc40fc
